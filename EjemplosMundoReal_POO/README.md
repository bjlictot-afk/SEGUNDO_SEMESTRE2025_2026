## Ejemplo del Mundo Real: Sistema de Estudiantes

Este programa fue desarrollado en Python utilizando Programación Orientada a Objetos (POO).
El ejemplo representa un sistema de estudiantes y cursos, modelando una situación del
mundo real mediante el uso de clases, atributos y métodos.

La clase Estudiante contiene la información básica de cada estudiante, como el nombre
y la carrera. La clase Curso permite gestionar un conjunto de estudiantes, demostrando
la interacción entre objetos.

Este ejemplo evidencia los principios fundamentales de la Programación Orientada a
Objetos, como la encapsulación y la organización del código, facilitando su comprensión
y mantenimiento.
